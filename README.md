## Description

Application web de gestion de bibliotheque universitaire developpee avec 
Spring Boot et PostgreSQL, en suivant la methodologie Agile Scrum.

Enseignant : Dr OUEDRAOGO Moise

---

## Fonctionnalites

- Recherche simple et avancee d'ouvrages
- Gestion des emprunts (max 3, 14 jours)
- Reservation avec file d'attente FIFO
- Calcul automatique des penalites de retard
- Notifications par email
- Generation de rapports statistiques

---

## Stack Technique

- Java 17
- Spring Boot 4.0.6
- PostgreSQL 16
- Thymeleaf + Bootstrap 5
- Spring Security + JWT
- JUnit 5 + Mockito
